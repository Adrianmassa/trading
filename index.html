A
主升段 · 主動進攻
大盤或族群多頭
標的明確主升段
籌碼持續偏多
允許加碼、攻擊性強
部位 60%～80%
B
震盪防守 · 降頻交易
盤勢不明 / 結構混亂
主升段不明確
已有戰果要守
只做 A+ setup
部位 20%～40%
A → B 切換
主升段結束、持股轉弱、族群退潮、波動加大
→ 縮部位、停加碼、提高出場敏感度
B → A 切換
至少3項成立：指數墊高、標的主升段、大戶連買、帳戶穩定、盤感清楚
→ 不能憑感覺切換
P1低檔強攻試單高
觸發條件
回測完成；RSI上穿40~50；MACD縮口或翻紅；大戶由賣轉買；未爆量追高
動作
先建立試單 30%
→試單進場
P2突破確認加碼高
觸發條件
突破關鍵價位；量能放大；結構站穩
動作
確認後再加 30%~40%
→加碼（非一開始就滿倉）
P3不符合條件 → 不進場高
觸發
低檔強攻與突破確認條件皆未滿足
動作
不進場，部位 0%
→觀察
P4B模式進場限制高
觸發
當前為 B 模式
限制
只允許最強訊號的新倉；總部位受B模式上限限制
→少量試單 / 不進
H1走對 → 續抱中
條件
價格結構未壞；量能正常；籌碼未轉空
動作
續抱，不變動部位
→續抱
H2走對且突破 → 加碼中
條件
已試單成功且突破確認成立
動作
依規則追加部位（只加走對的單）
→加碼
H3走弱 → 先減碼高
條件
高檔震盪轉弱；量縮不漲；籌碼背離
動作
先減 1/3 或依規則（不要等整體翻空才動）
→減碼
H4走錯 → 直接砍單高
條件
跌破結構支撐；籌碼連續轉空；風險事件發生
動作
直接減碼或出場（不能用希望代替紀律）
→出場 / 大減碼
R1 · 最高
收益鎖定線
獲利 ≥ +8%~10%
→ 直接先賣 1/2（50%）
R2 · 最高
RSI過熱鎖利
RSI(5) ≥ 85
→ 直接先賣 1/2（50%）
R3 · 最高
爆量衝高保命
單日漲幅 ≥ +7% 且成交量 ≥ 近5日均量1.8倍
→ 先賣 1/3（33%）
R4 · 最高
結構破壞止血
跌破關鍵支撐；大戶連續賣；長黑破位
→ 砍 30%~50%，必要時全出
R5 · 最高
單一持股上限
建立或加碼後，單一股票部位將超過總資金30%
→ 禁止再加碼；必要時調降至 ≤ 30%
R6 · 最高
禁止攤平虧損股
持股虧損中且結構未明顯轉強
→ 禁止加碼，只能觀察或減碼
R7 · 最高
禁止因成本低而不賣
觸發 R1~R4 任一條件時
→ 照規則執行（益航就是例子）
R8 · 最高
禁止情緒單
非計畫內標的；無完整setup；想補虧損
→ 不得下單，部位 0%
E1第一次停利高
觸發
R1 / R2 / R3 任一成立
動作
先賣 1/3~1/2（33%~50%）先保一段
→分批停利
E2第二次停利高
觸發
第一次停利後又出現過熱 / 背離 / 爆量轉弱
動作
再賣剩餘的一半（25%~35%）
→進一步停利
E3保留底倉中
條件
有中長線題材；整體盤勢仍偏多
動作
保留總持股 20% 以內底倉
→底倉續抱
E4防守性出清高
觸發
自高點回落 >5%；RSI跌回70以下
動作
至少再賣 1/3（補救第一賣點）
→補救性減碼
E5風險性出清高
觸發
自高點回落 >8%~10%；大戶/法人同步轉賣
動作
持股降到 1/3 以下，必要時全出
→防守出場
E6結構停損最高
觸發
跌破支撐、破頸線、週線轉空
動作
全部或大部分出場（不能硬撐）
→停損
